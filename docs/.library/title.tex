\begin{abstract}
Toxicity is a problem in the current gaming community, companies track and punish users according to their behaviour using their own detection algorithms.
This project analyzes the behaviour of users and streamers on the streaming site twitch.tv using a toxicity detection algorithm.
The influence users have on each other and the streamer was researched to see if toxicity spreads in communities. Unfortunately gathering in game data for popular games is difficult for privacy reasons, this is why this project focuses on the streaming site twitch.tv instead of actual in game chats as it is much more convenient to obtain a large dataset.
